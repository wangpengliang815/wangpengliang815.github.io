docker version

显示docker版本信息。

docker info

显示docker的系统信息，包括镜像和容器数量。

docker 命令 --help

帮助文档。

docker images

查看所有本地主机上的镜像。

标签 描述
TAG 镜像标签
IMAGE ID 镜像ID
CREATED 镜像创建时间
SIZE 镜像大小
REPOSITORY 镜像仓库源

可选项
-a, --all：列出所有镜像
-q, --quiet：只显示镜像ID

docker search

镜像搜索。

可选项

--filter STARS=3000： 搜索Stars数大于3000镜像

docker pull

下载镜像，默认下载最新版。

等价于

docker pull version

指定版本下载。

docker rmi

删除镜像。

docker run [可选参数] image

参数说明

  • --name： 容器名 用来区分容器
  • -d ：后台方式运行
  • -it  ：使用交互方式运行，进入容器查看内容
  • -p：指定容器的端口 -p  8080:8080
  • -P ：随机指定端口（大写-P）

测试启动容器

docker -ps

查看Docker当前运行的容器。

参数说明

  • -a ：列出当前在运行的容器+历史容器
  • -q ：只显示容器编号

exit

退出容器。

docker rm cId/cName

docker star cId/cName

启动容器。

docker restart cId/cName

重启容器。

docker stop cId/cName

停止当前正在运行的容器。

docker kill cId/cName

强制停止当前容器。

docker run -d cId/cName

后台运行容器。

docerk logs cId/cName

查看日志。

docker top cId/cName

查看容器进程信息。

docker inspect cId/cName

查看镜像的元数据新信息。

docker exec

docker attach

attach 与 exec 区别

  • docker exec： 进容器后开启一个新的后端，可以直接操作
  • docker attach：进入容器正在执行终端，不会启动新的进程

docker cp

文件拷贝，从docker内拷贝文件到宿主机。

apt-get update && apt-get install iputils-ping

在容器内安装 ping 工具。
