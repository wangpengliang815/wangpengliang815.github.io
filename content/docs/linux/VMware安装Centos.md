# 软件准备

- 软件：`VMware`
- 操作系统：`CentOS7`

# 虚拟机准备


打开 `VMware` 选择新建虚拟机

![](/images/2021-09-03-14-54-03.png)


典型安装与自定义安装

- 典型安装：`VMware` 会将主流的配置应用在虚拟机的操作系统上，对于新手来很友好
- 自定义安装：自定义安装可以针对性的把一些资源加强，把不需要的资源移除。避免资源浪费



**这里选择典型安装**

![](/images/2021-09-03-14-54-59.png)

**选择稍后安装操作系统**

![](/images/2021-09-03-14-55-14.png)

**选择操作系统为Linux**



![](/images/2021-09-03-14-55-43.png)

**输入虚机名称和存储位置**

![](/images/2021-09-03-14-55-58.png)

**设置虚机磁盘大小**

![](/images/2021-09-03-14-56-11.png)

**自定义硬件**

![](/images/2021-09-03-14-56-26.png)

![](/images/2021-09-03-14-56-53.png)

根据自己电脑实际情况分配

- 内存：2G
- 处理器数量：2 / 内核数量：2
- 网络适配器：NAT 模式
- 移除打印机和声卡



# Centos安装

在创建好的虚机上右键设置

![](/images/2021-09-03-14-57-26.png)

开启虚拟机

![](/images/2021-09-03-14-57-39.png)

安装操作系统
开启虚拟机后会出现以下界面:


- `Install CentOS 7` 安装`CentOS 7`
- `Test this media & install CentOS 7` 测试安装文件并安装CentOS 7
- `Troubleshooting` 修复故障

选择第一项，安装CentOS 7，回车进入下面的界面

![](/images/2021-09-03-14-57-54.png)

选择安装过程中使用的语言

![](/images/2021-09-03-14-58-08.png)

安装位置及分区设置；这里分区使用自动配置

![](/images/2021-09-03-14-58-32.png)

软件选择：最小安装

![](/images/2021-09-03-14-59-04.png)

网络和主机名：设置主机名称开启网络连接

![](/images/2021-09-03-14-59-41.png)

开始安装

![](/images/2021-09-03-15-00-03.png)

设置`Root`密码并创建新的用户`admin`

![](/images/2021-09-03-15-00-26.png)

等待安装完成重启系统即可。
